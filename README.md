# Voxelbulb

Elite Mandelbulb simulator 🦾

Comes with free post-processing shaders (hotkey: backslash):
- Edge-detect 📐
- Dithering 🎲

### Run w/ CLI:
```shell
pip install -r requirements.txt
python graphics_mdb.py
```
