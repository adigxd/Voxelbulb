# Voxelbulb

Elite [Mandelbulb](https://en.wikipedia.org/wiki/Mandelbulb) simulator

Comes with free post-processing shaders (hotkey: backslash):
- Edge-detection 📐
- Dithering 🎲

### Run w/ CLI:
```shell
pip install -r requirements.txt
python graphics_mdb.py
```
